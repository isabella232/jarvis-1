# clean-css-types

```
clean-css-types <path> [args]

Find and remove declarations for style (css, scss, etc.) files

Positionals:
  path

Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -e, --ext                                          [array] [default: [".css"]]
```
